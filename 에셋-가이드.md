# 붕어빵 타이쿤 — 이미지 에셋 가이드

게임 파일과 같은 위치에 `assets` 폴더를 만들고 아래 파일명으로 이미지를 넣으면, 게임이 자동 감지해서 SVG 대신 사용합니다. 파일이 없으면 기존 SVG 그래픽으로 동작하므로 일부만 넣어도 됩니다.

## 필요한 파일 목록

| 파일명 | 용도 | 권장 크기 | 필수 조건 |
|---|---|---|---|
| `fish_raw.png` | 반죽 상태(굽기 전) 붕어빵 | 480×304 | 투명 배경, 3개 세트 |
| `fish_gold.png` | 노릇하게 익은 붕어빵 | 480×304 | 위와 동일 구도·크기 |
| `fish_burnt.png` | 탄 붕어빵 | 480×304 | 위와 동일 구도·크기 |
| `customer_1.png` ~ `customer_8.png` | 손님 캐릭터 (있는 만큼 사용) | 360×408 | 투명 배경, 정면 |
| `city_bg.png` | 거리 배경 (게임+타이틀 공용) | 960×460 | 하단이 지면 |

주의사항: 붕어빵 3종은 반드시 같은 구도·같은 크기로 만들어야 합니다(익는 과정이 세 이미지의 크로스페이드로 표현되기 때문). 캐릭터는 발밑 여백을 최소화하면 바닥에 잘 붙어 보입니다.

## AI 이미지 생성 프롬프트

미드저니, DALL-E, 스테이블 디퓨전 등에 사용할 수 있는 프롬프트입니다. 캐릭터·소품은 `--no background` 또는 "transparent background, isolated on white" 를 붙이고, 생성 후 배경 제거 도구(remove.bg 등)로 투명화하세요.

### 붕어빵 3종 (같은 시드/구도 유지 권장)

```
cute kawaii taiyaki fish-shaped pastry, facing right, waffle grid texture,
smiling face with big eye, thick brown outline, mobile game item asset,
pastel cartoon style, soft shading, glossy highlight, isolated on white background
```

- raw: 위 프롬프트 + `pale cream uncooked batter color`
- gold: 위 프롬프트 + `golden brown perfectly baked, warm glow`
- burnt: 위 프롬프트 + `dark charred burnt, small smoke wisps, sad face`

### 손님 캐릭터 (8종 각각 변형)

```
chibi character with oversized round head, small body, winter outfit with
beanie hat, thick outline, blush cheeks, sparkling eyes, front facing,
cozy pastel mobile game style like Animal Crossing pocket camp,
full body, isolated on white background
```

변형 아이디어: 판다 캐릭터, 주황 고양이, 분홍 돼지, 곰, 할아버지, 어린이, 목도리 두른 여성, 야구모자 소년 등. "chibi panda character in winter coat..." 처럼 주어만 바꾸면 됩니다.

### 거리 배경

```
pastel colored city street in winter, cute cartoon buildings with striped awnings
and glowing windows, light snow, street lamps, soft daylight, warm cozy mood,
mobile game background art, wide shot, no characters
```

## 무료 에셋 대안

AI 생성 대신 무료 에셋을 쓰려면 Kenney(kenney.nl)나 itch.io의 CC0 게임 에셋에서 캐릭터/음식 팩을 받아 위 파일명으로 저장하면 됩니다.

## 적용 방법

1. `Bungeoppang Tycoon` 폴더 안에 `assets` 폴더 생성
2. 위 파일명으로 PNG 저장
3. 게임 새로고침 — 자동 적용

이미지를 준비해서 폴더에 넣어주시면, 크기 조정·위치 보정·연출 개선은 제가 이어서 다듬어 드릴 수 있습니다.
